* 用况描述
  - 发布通知
    - 用况名称 :: 发布通知
    - 参与的执行者 :: 用户
    - 前置条件 ::
         一个合法的用户已经登入系统
    - 事件流 ::
      - 基本路径
        1. 用户选择要发布通知的活动或任务
           1) 如果用户没有发布相应通知的权限，用况结束
           2) 如果用户有相应权限，显示发布通知按钮
        2. 当用户点击「发布通知」，用况开始
        3. 用户输入通知时间
        4. 用户输入通知内容
        5. 用户选择通知受众
        6. 用户提交通知请求
           1) 显示发布成功，用况结束
           2) 显示发布失败，用况结束
      - 可选路径
        - 在 3-6 中任一步，用户均能退出发布流程，用况结束
